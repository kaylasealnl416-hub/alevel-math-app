## 📋 Week 1：微信OAuth集成

### Day 1-2：微信开放平台配置

**任务**：
- [ ] 注册微信开放平台账号
- [ ] 创建网站应用
- [ ] 获取AppID和AppSecret
- [ ] 配置授权回调域名
- [ ] 测试微信扫码登录流程

**技术要点**：
- 微信OAuth 2.0协议
- 授权码模式
- 获取用户信息

### Day 3-4：后端认证系统

**任务**：
- [ ] 创建认证路由（`/api/auth`）
- [ ] 实现微信登录接口
- [ ] JWT Token生成与验证
- [ ] 刷新Token机制
- [ ] 认证中间件

**API端点**：
```
POST /api/auth/wechat - 微信登录
POST /api/auth/refresh - 刷新Token
GET /api/auth/me - 获取当前用户
POST /api/auth/logout - 登出
```

### Day 5-7：前端登录界面

**任务**：
- [ ] 创建登录页面组件
- [ ] 微信扫码组件
- [ ] Token管理（localStorage）
- [ ] 登录状态管理（Zustand）
- [ ] 路由守卫（保护需要登录的页面）
