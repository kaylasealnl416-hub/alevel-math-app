# 问题清单 2026-03-22 用户测试反馈

## P0 — 功能崩溃

### 1. ✅ Wrong Answers 页面崩溃：`t.options.map is not a function`
- **页面**: `/wrong-questions`
- **原因**: QuestionCard 和 ExamResultPage 用 `.map()` 遍历 options，但 AI 生成的题目 options 是对象 `{A:"...", B:"..."}` 不是数组
- **修复**: QuestionCard.jsx 和 ExamResultPage.jsx 改用 `Array.isArray()` 判断 + `Object.entries()` 兼容对象格式

### 2. ✅ 考试提交失败：「考试已结束，无法保存答案」
- **页面**: `/exams/18/result`
- **现象**: alert 弹窗 "Submission failed: 考试已结束，无法保存答案"，然后卡在 "Loading exam results..."
- **原因**: 时间到后自动提交，但可能触发了重复提交（状态已变为 submitted/graded），结果页加载不出来
- **修复**:
  - 后端 submitExam/autoSubmitExam/saveAnswer：已提交状态返回 success 而非抛错
  - 前端 handleAutoSubmit：无论提交成功失败都跳结果页
  - 前端 handleConfirmSubmit：遇到"已提交"错误直接跳结果页，其他错误用 Toast.error

### 3. ✅ Exams 列表页无限 loading
- **页面**: `/exams`
- **现象**: 一直显示 "Loading exams..." 转圈
- **修复**: 超时从 30s 增到 60s；5秒后显示 "Server is waking up, please wait..."；请求失败后显示错误+重试按钮
