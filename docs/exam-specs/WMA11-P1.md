## 试卷结构规律

- 题目连续编号 Q1–Q11，无 Section 分区
- 每题含 2–5 个小问，各小问 1–6 分
- Q1–Q4：基础题（共约20分）
- Q5–Q8：中等题（共约30分）
- Q9–Q11：综合题（共约25分）
- 最后2–3题通常为微积分综合 + 曲线面积题

| 章节ID | 章节名称 | 典型题型 |
|--------|---------|---------|

## 出题规则（倒推法）

1. 先选整数/简单分数答案 → 反推题目参数
2. 勾股数：(3,4,5)(5,12,13)(8,15,17) 用于圆/距离题
3. 特殊角：30°/45°/60° 确保三角值整洁
4. 每题小问递进：(a) 求基础量 → (b) 利用(a)结果推导 → (c) 应用/综合
5. Mark scheme：M（方法分）/ A（精度分）/ B（独立分）
