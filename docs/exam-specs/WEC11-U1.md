## Mark Scheme 规范

### MCQ 格式
- 情境：1-2句真实场景（国家+数据+年份）
- 4个选项（A/B/C/D），每个错误选项需附 K/A 级解释

### 4分题（Section B）格式：K1 + A1 + AN2
- K1（1分）：定义/公式（1句）
- A1（1分）：引用题目情境（1句）
- AN2（2分）：两步因果链（2-3句）
- 总长度：约60-80词

### 14分题（Section C e小问）格式：KAA(8分) + EV(6分)
- Level 3 KAA：两个完整分析链，双向考虑
- Level 3 EV：平衡评价，有条件支撑的明确结论

### 20分题（Section D）格式：KAA(12分) + EV(8分)
- Level 4 KAA：全面深入，数据支撑，多层次
- Level 3 EV：平衡评价 + 明确结论
- 约300-350词
