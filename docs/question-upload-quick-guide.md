# PDF/Word 题库上传功能 - 快速使用指南

## 🚀 快速开始

### 1. 启动服务

**后端**（已启动）:
```bash
cd backend
bun run dev
# ✅ 运行在 http://localhost:4000
```

**前端**（已启动）:
```bash
bun run dev
# ✅ 运行在 http://localhost:3000
```

### 2. 访问上传页面

打开浏览器访问：
```
http://localhost:3000/questions/upload
```

**注意**: 需要先登录才能访问上传页面

---

## 📝 使用步骤

### Step 1: 登录系统

1. 访问 http://localhost:3000/login
2. 使用测试账号登录：
   - 邮箱: `student1@test.com`
   - 密码: `test123`

### Step 2: 进入上传页面

1. 登录后，访问 http://localhost:3000/questions/upload
2. 或者在导航栏中找到"题库上传"链接

### Step 3: 上传文件

**方式 1: 拖拽上传**
- 将 PDF 或 Word 文件拖拽到上传区域

**方式 2: 点击选择**
- 点击"选择文件"按钮
- 在文件选择器中选择文件

**支持的格式**:
- PDF (.pdf)
- Word (.doc, .docx)

**文件大小限制**: 最大 50MB

### Step 4: 等待解析

1. 点击"开始上传并解析"按钮
2. 系统会显示进度条
3. AI 正在提取题目（可能需要 10-60 秒）

### Step 5: 审核题目

解析完成后，会自动进入审核界面：

1. **查看提取的题目**
   - 题目内容
   - 选项（如果是选择题）
   - 答案
   - 解释
   - 置信度（AI 识别的准确性）

2. **编辑题目**（如果需要）
   - 点击任意字段进行编辑
   - 修改题目内容、选项、答案等
   - 调整难度（1-5）
   - 添加或修改标签

3. **删除题目**（如果不需要）
   - 点击题目右上角的"删除"按钮

### Step 6: 选择章节

在页面顶部选择题目所属的章节：
- Pure Mathematics 1
- Pure Mathematics 2
- Statistics
- Mechanics

### Step 7: 批量保存

1. 确认所有题目无误
2. 点击"批量保存 X 个题目"按钮
3. 等待保存完成
4. 查看保存结果

## 💡 使用技巧

### 1. 文档格式建议

为了获得最佳识别效果，建议文档格式如下：

```
题目编号
题目内容

A. 选项1
B. 选项2
C. 选项3
D. 选项4

答案: B
解释: 答案解释
```

### 2. 数学公式

AI 会自动识别数学公式并转换为 LaTeX 格式：
- 输入: `x^2 + 5 = 13`
- 输出: `$x^2 + 5 = 13$`

### 3. 置信度说明

- **绿色（90%+）**: 识别准确，可以直接保存
- **橙色（70-90%）**: 建议检查一下
- **红色（<70%）**: 需要仔细审核

### 4. 批量编辑

如果有多个题目需要修改相同的字段（如章节、难度），可以：
1. 先审核完所有题目
2. 统一选择章节
3. 批量保存

## ⚠️ 常见问题

### Q1: 上传失败怎么办？

**可能原因**:
- 文件格式不支持（只支持 PDF、Word）
- 文件太大（超过 50MB）
- 网络问题

**解决方法**:
- 检查文件格式和大小
- 刷新页面重试
- 查看浏览器控制台错误信息

### Q2: AI 提取不准确怎么办？

**解决方法**:
- 在审核界面手动编辑
- 删除不需要的题目
- 调整文档格式后重新上传

### Q3: 保存失败怎么办？

**可能原因**:
- 未选择章节
- 题目格式不完整
- 数据库连接问题

**解决方法**:
- 确保选择了章节
- 检查必填字段（题目内容、答案）
- 查看后端日志

### Q4: 进度卡住不动怎么办？

**解决方法**:
- 等待 1-2 分钟（大文件需要更长时间）
- 刷新页面重试
- 检查后端服务器是否正常运行

## 🎯 最佳实践

1. **文档准备**
   - 使用清晰的题目编号
   - 选项使用 A/B/C/D 标识
   - 明确标注答案和解释

2. **分批上传**
   - 建议每次上传 10-20 页
   - 避免单次上传过大文件
   - 分批审核更高效

3. **及时审核**
   - 上传后立即审核
   - 不要积累太多待审核题目
   - 保持题库质量

4. **备份原文件**
   - 保留原始文档
   - 方便后续修正
   - 避免数据丢失
