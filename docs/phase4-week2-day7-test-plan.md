## 🧪 测试场景

### 场景 1：完整答题流程
1. 进入考试列表页面
2. 点击"创建新考试"（或选择进行中的考试）
3. 进入答题页面
4. 依次回答所有题目
5. 标记几道题目
6. 使用答题卡跳转
7. 提交考试
8. 查看结果

**预期结果**：
- 所有步骤流畅无阻
- 答案正确保存
- 提交成功
- 跳转到结果页面

### 场景 2：中断恢复流程
1. 开始答题
2. 回答部分题目
3. 关闭浏览器或刷新页面
4. 重新进入答题页面

**预期结果**：
- 之前的确恢复
- 可以继续答题
- 计时器继续计时

### 场景 3：超时自动提交
1. 创建一个时间限制很短的考试（如1分钟）
2. 开始答题
3. 等待时间到

**预期结果**：
- 计时器倒计时正确
- 剩余时间 < 5分钟时显示警告
- 时间到时自动提交
- 显示提示信息
- 跳转到结果页面

### 场景 4：防作弊检测
1. 开始答题
2. 切换到其他标签页
3. 切换回答题页面
4. 点击浏览器外部区域（失焦）
5. 完成答题并提交

**预期结果**：
- 切换标签页和失焦事件被记录
- 不影响正常答题
- 事件数据保存到服务器

### 场景 5：自动保存
1. 开始答题
2. 回答一道题目
3. 等待30秒
4. 观察自动保存指示器

**预期结果**：
- 30秒后显示"保存中..."
- 保存成功后指示器消失
- 刷新页面后答案恢复
