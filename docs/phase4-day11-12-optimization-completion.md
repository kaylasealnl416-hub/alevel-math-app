#### 发现的问题

**问题 1：考试状态不一致**
- **现象**：提交后状态为 `submitted` 而非 `graded`
- **原因**：批改流程可能异步执行
- **影响**：部分依赖 `graded` 状态的功能失败
- **建议**：优化批改流程，确保同步完成

**问题 2：统计数据为 null**
- **现象**：`totalScore`, `maxScore` 等字段为 null
- **原因**：批改未完成或数据未更新
- **影响**：无法显示完整成绩
- **建议**：检查批改服务的数据更新逻辑

**问题 3：考试列表查询失败**
- **现象**：获取考试列表返回错误
- **原因**：可能是权限或查询条件问题
- **影响**：无法查看历史考试
- **建议**：检查 API 权限和查询逻辑
