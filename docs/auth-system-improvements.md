## 📝 使用说明

### 注册新用户
1. 访问 `/register`
2. 填写昵称、邮箱、密码
3. 选择年级（AS / A2）
4. 查看密码强度提示
5. 确认密码
6. 点击注册

### 登录
1. 访问 `/login`
2. 输入邮箱和密码
3. 勾选"记住我"（可选）
4. 点击登录

### 更新个人信息
1. 点击导航栏的用户头像
2. 修改昵称、年级、目标大学、手机号
3. 点击"保存更改"

### 退出登录
1. 点击导航栏的"登出"按钮
2. 或在个人信息页面点击"退出登录"
