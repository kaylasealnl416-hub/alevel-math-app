## 🚀 核心算法

### 1. 推荐算法

```javascript
正确率 < 30%  → Critical (优先级 5) → 章节复习
正确率 30-60% → Weak (优先级 4) → 章节复习
正确率 60-80% → Practice (优先级 3) → 练习题
有错题       → Review (优先级 5) → 错题复习
```

### 2. 学习计划分配

```javascript
每日任务数 = Math.ceil(总任务数 / 计划天数)
任务排序 = 按优先级降序
分配策略 = 均匀分配到每一天
```

### 3. 批改引擎

```javascript
客观题 → 直接比对答案
主观题 → AI 批改（未来实现）
统计分析 → 难度/题型/知识点
```
