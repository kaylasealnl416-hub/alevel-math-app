**示例**:
```
"AD曲线向右下方倾斜的原因：
1. 财富效应：价格下降 → 实际财富增加 → 消费增加
2. 利率效应：价格下降 → 货币需求减少 → 利率下降 → 投资增加
3. 国际贸易效应：价格下降 → 出口增加，进口减少"
```
