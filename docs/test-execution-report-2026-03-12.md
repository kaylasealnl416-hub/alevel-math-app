## 📝 测试脚本示例

### 创建综合测试脚本

```bash
# backend/run-all-tests.sh
#!/bin/bash

echo "🧪 运行所有测试..."
echo ""

# 1. 认证测试
echo "📝 1. 用户认证测试"
bun run tests/auth-flow.test.js
if [ $? -ne 0 ]; then
  echo "❌ 认证测试失败"
  exit 1
fi
echo ""

# 2. 考试流程测试
echo "📝 2. 考试流程测试"
bun run tests/exam-workflow.test.js
if [ $? -ne 0 ]; then
  echo "❌ 考试流程测试失败"
  exit 1
fi
echo ""

# 3. API 测试
echo "📝 3. API 集成测试"
bun run src/test-exams-api.js
if [ $? -ne 0 ]; then
  echo "❌ API 测试失败"
  exit 1
fi
echo ""

echo "✅ 所有测试通过！"
```

### 添加到 package.json

```json
{
  "scripts": {
    "test": "bun run run-all-tests.sh",
    "test:auth": "bun run tests/auth-flow.test.js",
    "test:exam": "bun run tests/exam-workflow.test.js",
    "test:api": "bun run src/test-exams-api.js",
    "test:watch": "bun test --watch"
  }
}
```
