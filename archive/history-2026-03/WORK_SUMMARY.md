## 🚀 如何使用

### 启动服务

```bash
# 后端
cd backend
bun run dev

# 前端
bun run dev
```

### 访问应用

- **主应用**：http://localhost:3000
- **API测试**：http://localhost:3000?test=api
- **后端API**：http://localhost:4000

### 切换数据源

编辑 `.env.local`：
```bash
# 本地数据模式
VITE_USE_API=false

# API数据模式
VITE_USE_API=true
```
